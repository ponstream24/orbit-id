# @orbit-id/core

Reference implementation of Orbit ID v1 for the monorepo.

## API

```ts
import {
  encode,
  parse,
  generate via OrbitGenerator,
  isValid,
} from "@orbit-id/core";
```

See [Library API](../../docs/en/library-api.md) and fixtures in [`spec/conformance/`](../../spec/conformance/).
